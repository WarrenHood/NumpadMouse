# NumpadMouse

Simulate your mouse input using your keyboard!

Controls:
    Mouse Left: Numpad 4
    Mouse Right: Numpad 6
    Mouse Up: Numpad 8
    Mouse Down: Numpad 5
    Left Click: Numpad 0
    Right Click: Numpad 1
    Decrease Mouse Speed: Numpad 2
    Increase Mouse Speed: Numpad 3
    Slow Aim: Left Control
    Terminate Aim Thread: Numpad period(.)
